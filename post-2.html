Top 10 thesis mistakes and how to fix them

Avoid common pitfalls that delay submission. Here are quick fixes you can apply now.

  1. Poor research question: Fix: narrow scope and define measurable variables.
  2. Weak literature review: Fix: map core themes, use recent sources, synthesise not summarize.
  3. Vague methodology: Fix: specify design, sample size calculations, instruments, and analysis steps.
  4. Data quality issues: Fix: pilot test instruments, check for missing data, and document cleaning steps.
  5. Poor structure & flow: Fix: use clear chapter mapping and signpost transitions between sections.
  6. Inconsistent referencing: Fix: pick a style and run a reference manager (Zotero/EndNote).
  7. Overlong results/underdeveloped discussion: Fix: keep results factual, interpret in discussion, tie to questions.
  8. Poor figures & tables: Fix: clear labels, readable fonts, and descriptive captions.
  9. Not preparing for viva: Fix: mock vivas, prepare slides and short summary of contribution.
  10. Plagiarism & ethics oversight: Fix: check similarity reports, revise, and document approvals.

Final advice

Address issues early by using checklists, draft reviews and milestone-based feedback. If you need help fixing these areas, get in touch for a targeted plan.
